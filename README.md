## ✨ المميزات والخصائص الرئيسية:

1. 🎬 **عرض سينمائي فاخر للصور (Ken Burns Effect):**
   - مدة الشريحة 12 ثانية كاملة لراحة المشاهدة.
   - دعم عرض صورتين متتاليتين في الشريحة الواحدة (Dual-Photo Transition) أو صورة واحدة مع زوم سلس.

2. 🎵 **مشغل قائمة أغاني متتابعة (Sequential Playlist Engine v4.0):**
   - تشغيل أغاني يوتيوب متتالية أوتوماتيكياً وبدون أي إعلانات (`youtube-nocookie`).
   - دعم رفع وتشغيل ملفات MP3 محلياً أو عزف لحن عيد الميلاد المدمج تلقائياً.

3. ☁️ **تخزين سحابي مجاني ومستقر (GitHub Data Engine):**
   - قراءة قائمة الأغاني والتهاني مجاناً لكل الزوار من المستودع.
   - حفظ التعديلات بضغطة زر واحدة 🟢 **"حفظ للجميع أونلاين"** بعد إدخال GitHub Token مرة واحدة فقط (حفظ تلقائي بدون نوافذ منبثقة).

4. 🎂 **كعكة تفاعلية وشمعة قابلة للإطفاء:**
   - اضغط لإطفاء الشمعة مع إطلاق زينة وألعاب نارية (Confetti) وألحان احتفالية.

5. 💌 **دفتر التهاني والأمنيات العائلي (Wishes Board):**
   - تسجيل تبريكات الأهل والأقارب وتثبيتها بشكل دائم.

6. 📱 **تجاوب كامل وتصميم ملكي:**
   - ألوان وأيقونات ذهبية وفخمة متوافقة مع أجهزة الأيفون، الأندرويد، والكمبيوتر.

## 📸 كيفية إضافة وتحديث الصور والأغاني:

1. **الصور:**
   - ضع صور الأميرة نَفَس في مجلد `images/` بالأسماء التالية: `nafas1.jpg` حتى `nafas7.jpg`.
   - يمكنك استخدام زر **"اختيار الصور محلياً"** لمعاينة الصور فوراً من الهاتف أو الكمبيوتر.

2. **الموسيقى:**
   - افتح الموقع عبر رابط التعديل: `https://karim-fayed.github.io/NafasOneYears/?edit=1`
   - الصق رابط أغنية يوتيوب واضغط **"إضافة للقائمة"** أو **"تشغيل الآن"**.
   - اضغط 🟢 **"حفظ للجميع أونلاين"** لتثبيت الأغاني لكل أفراد العائلة.
