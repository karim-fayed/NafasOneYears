# 👑 احتفالية عيد الميلاد الأول - للأميرة نَفَس 💕

صفحة تذكارية وسينمائية بمناسبة اكتمال العام الأول للأميرة **نَفَس كريم**.

## 🌐 رابط الصفحة أونلاين:
👉 **[https://karim-fayed.github.io/NafasOneYears/](https://karim-fayed.github.io/NafasOneYears/)**

---

## 📸 كيفية إضافة صور نَفَس:

1. افتح مجلد `images` في هذا المستودع.
2. قم برفع 7 صور للأميرة نَفَس بالأسماء التالية:
   - `nafas1.jpg` (الصورة الأولى)
   - `nafas2.jpg` (الصورة الثانية)
   - `nafas3.jpg` (الصورة الثالثة)
   - `nafas4.jpg` (الصورة الرابعة)
   - `nafas5.jpg` (الصورة الخامسة)
   - `nafas6.jpg` (الصورة السادسة)
   - `nafas7.jpg` (الصورة السابعة)
3. قم بوضع ملف الأغنية بصيغة MP3 في المجلد الرئيسي باسم `birthday.mp3`.
4. اعمل `git commit` و `git push` لتحديث الصور مباشرة على الرابط الأونلاين!

---

## ✨ المميزات والخصائص:
- 🎬 **عرض سينمائي كالفيديو (Ken Burns Effect)** للصور مع تحكم كامل بالتشغيل والإيقاف والتنقل.
- 🎵 **عازف موسيقي ذكي (Web Audio Synth)** يعزف أغنية Happy Birthday تلقائياً بدون الحاجة لملفات خارجية + إمكانية رفع أغنيتك الخاصة MP3.
- 🎂 **كعكة تفاعلية مع شمعة قابلة للإطفاء** مع مؤثرات ألعاب نارية وConfetti.
- 💌 **دفتر تهاني عائلي تفاعلي** لتسجيل تبريكات الأهل والأقارب.
- 📲 **أزرار مشاركة فورية على واتساب وفيس بوك** مع رسالة معنونة ومجهزة للنشر.
